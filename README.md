# 在线便签 - 免费便捷的在线笔记工具

## 项目介绍

在线便签是一款基于 Vue 3 + Vite 开发的免费、便捷的在线笔记工具，支持快速创建、编辑和管理便签，数据本地存储，安全可靠。

- **数据本地存储**：所有数据存储在浏览器本地，无需服务器，安全可靠
- **响应式设计**：适配各种设备屏幕尺寸，支持移动端访问
- **多语言支持**：内置中文和英文支持
- **深色模式**：支持亮色/深色模式切换，保护视力
- **PWA 支持**：可安装为本地应用，支持离线访问

## 安装与使用

### 本地开发

1. **克隆项目**
   ```bash
   git clone https://github.com/solfKwolf/text.git
   cd text
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

4. **构建项目**
   ```bash
   npm run build
   ```

### 生产部署

项目使用 GitHub Pages 部署，构建后的文件会自动发布到 `https://solfkwolf.github.io/text/`。
