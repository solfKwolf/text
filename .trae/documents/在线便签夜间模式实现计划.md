# 在线便签夜间模式实现计划

## 实现目标
1. 添加夜间模式切换按钮，支持正常/夜间模式切换
2. 页面右下角有圆角按钮，支持切换夜间模式/正常模式，中间显示不同图标
3. 夜间模式下，背景为黑色，note组件背景颜色为暗色为主
4. 自定义样式和背景在夜间模式无效

## 实现步骤

### 1. 扩展样式变量系统
**修改文件：** `src/styles/variables.css`
- 添加暗色主题变量（--dark-bg-color, --dark-note-bg, --dark-text-color等）
- 确保所有颜色使用CSS变量，便于主题切换

### 2. 实现主题切换功能
**修改文件：** `src/App.vue`
- 添加主题状态管理（light/dark）
- 在右下角添加圆角切换按钮，使用CSS实现太阳/月亮图标
- 实现按钮点击事件切换主题
- 添加主题切换的平滑过渡效果

### 3. 修改全局样式
**修改文件：** `src/styles/common.css`
- 添加主题切换的基础样式
- 实现body和容器的暗色主题样式
- 添加平滑过渡动画

### 4. 调整Note组件样式
**修改文件：** `src/components/Note.vue`
- 修改noteStyle计算属性，根据主题状态应用不同样式
- 夜间模式下禁用自定义背景色，使用统一暗色
- 调整样式面板在夜间模式下的显示效果
- 确保文本颜色在夜间模式下清晰可读

## 技术要点
- 使用CSS变量实现主题切换
- 通过data属性控制主题状态
- 利用计算属性动态应用样式
- 添加平滑过渡动画提升用户体验
- 遵循Vue 3最佳实践，使用Composition API

## 预期效果
- 右下角显示圆角切换按钮，使用CSS绘制的太阳/月亮图标
- 点击按钮切换图标并切换主题模式
- 切换到夜间模式时，背景变为黑色，便签变为统一暗色
- 夜间模式下便签自定义颜色失效，使用默认暗色
- 所有过渡效果平滑自然

## 实现难度
简单，主要涉及CSS变量、条件样式应用和Vue 3状态管理，无复杂逻辑

## 遵循的Vue 3最佳实践
- 使用Composition API管理状态
- 组件样式隔离（scoped CSS）
- 使用CSS变量实现主题切换
- 添加适当的过渡动画
- 代码组织清晰，逻辑分离
- 响应式设计，适配不同设备
