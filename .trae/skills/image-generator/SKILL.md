# 图片生成器

## 功能介绍

这个技能用于生成各种图片资源，特别是暗黑模式图标，支持多种图片类型和生成规则。

### 核心功能

1. **暗黑模式图标生成**：根据浅色图标生成对应的暗黑模式图标
2. **图标样式转换**：支持不同样式之间的转换
3. **尺寸调整**：调整图片尺寸和分辨率
4. **格式转换**：转换图片格式（PNG、JPEG、SVG等）
5. **批量生成**：支持批量生成多张图片

## 支持的图片类型

- ✅ 图标（ICO、PNG、SVG）
- ✅ 背景图片
- ✅ 按钮和UI元素
- ✅ 暗黑模式变体
- ✅ 响应式图片

## 使用示例

### 生成暗黑模式图标

```javascript
// 示例：生成暗黑模式图标
const darkModeIcon = generateDarkModeIcon(lightIcon, options);
```

### 批量生成图标变体

```javascript
// 示例：批量生成不同尺寸的图标
const iconVariants = generateIconVariants(originalIcon, {
  sizes: [16, 24, 32, 48],
  formats: ['png', 'svg'],
  darkMode: true
});
```

## 生成规则

### 暗黑模式图标生成规则

1. **颜色转换**：将浅色转为深色，深色转为浅色
2. **对比度调整**：确保暗黑模式下的可读性
3. **保持风格**：保持原图标风格和细节
4. **优化显示**：针对暗黑背景优化图标显示效果

### 图标尺寸生成规则

1. **等比例缩放**：保持图标比例
2. **清晰度优化**：确保缩放后清晰
3. **格式适配**：根据尺寸选择合适的格式

## 实现方式

### 1. 基于现有图片生成

- 分析原始图片的颜色、形状和结构
- 根据生成规则转换为目标样式
- 优化输出图片的质量和大小

### 2. 基于描述生成

- 根据文字描述生成图片
- 支持详细的样式描述
- 支持多种输出格式

## 输入格式

### 图片输入

- 支持PNG、JPEG、SVG等常见格式
- 支持本地文件和URL

### 文字描述输入

```
描述：一个简约的月亮图标，用于暗黑模式切换，圆形，线条风格，大小24x24，PNG格式
```

## 输出格式

- PNG：透明背景，支持各种尺寸
- JPEG：适合背景图片
- SVG：矢量格式，支持无限缩放
- ICO：适合图标文件

## 最佳实践

1. **提供清晰的描述**：详细描述图片的样式、尺寸和用途
2. **使用高质量输入**：输入图片质量越高，生成结果越好
3. **指定输出格式**：明确指定需要的输出格式
4. **考虑不同设备**：为不同设备生成合适尺寸的图片
5. **测试生成结果**：在实际场景中测试生成的图片

## 示例应用

### 1. 暗黑模式切换图标

```
输入：太阳图标（PNG，24x24）
输出：月亮图标（PNG，24x24，暗黑模式）
```

### 2. 响应式图标集

```
输入：原始图标（SVG）
输出：
- 16x16 PNG
- 24x24 PNG
- 32x32 PNG
- 48x48 PNG
- 24x24 SVG（暗黑模式）
```

### 3. 自定义图标生成

```
输入："一个蓝色的文件夹图标，扁平化设计，圆角，24x24，PNG格式"
输出：蓝色文件夹图标（PNG，24x24）
```

## 性能考虑

- 大尺寸图片生成可能需要较长时间
- 批量生成建议限制数量
- SVG格式生成速度快，适合复杂图标
- PNG格式适合简单图标和UI元素

## 扩展功能

- 支持主题色生成
- 支持渐变和阴影效果
- 支持动画生成
- 支持3D效果
